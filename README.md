# Singularity Skill

Portable full-platform agent skill for x402 Studio and Singularity Layer.

## Install

```bash
npx skills add https://github.com/ivaavimusic/singularity-skill --skill singularity
```

## Repository Layout

- `singularity/` — installable skills.sh package
